## Essential linux commands

- Linux commands are case sensitive.
- Most of the linux commands are in lowercase. 
- `pwd` represents present working directory.
- `cd` represents change of directory.
- `ls` lists the directory contents.
  - `ls -l` represents directory contents in long listing format in lexicographical order. 
  - `ls -lr` represents in reverse lexicographical order.
  - `ls -lrt` represents in reverse lexicographical order with latest at the bottom.
  - `ls -lrth` represents in reverse lexicographical order with latest at the bottom and human readable format of sizes like MB instead of bytes.
- `cat` is to concatenate and display file contents. 
  - Usage of the command is `cat \etc\shells`. The contents of this file repreents the shells that are supported on this linux machine.
```
[root@symphony-dev-cluster1 ~]# cat /etc/shells
/bin/sh
/bin/bash
/usr/bin/sh
/usr/bin/bash
/usr/bin/tmux
/bin/tmux
/usr/bin/zsh
/bin/zsh
/bin/ksh
/bin/rksh
/usr/bin/ksh
/usr/bin/rksh
```  
- `clear` command clears the screen.
- `man` command displays the online manual for the command. `man` is a shortcut for manual.
  - `space` displays the next page during the display of `man`.
  - `Enter` to move line by line.
  - `g` to move to the top of the page, `G` to move to the bottom of the page.
  - `q` for quitting `man` page.
```
[root@symphony-dev-cluster1 ~]# man cat
```
